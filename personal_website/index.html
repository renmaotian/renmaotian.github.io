  • Excellent SEO performance

  • Stable with strong infrastructure support
  • Global CDN ensures low webpage latency
  • Increases professionalism and credibility in the tech community

How to create a GitHub.io personal website?

  1. Register a GitHub account, preferably using your name as the username.
  2. Create a new repository named username.github.io. Replace username with your name.
  3. After creating the repository, you'll be prompted to create or upload files.
  4. Create a file called index.html locally, save your HTML-formatted webpage content in it, then upload it and click "Commit changes".
  5. After a few minutes, you'll be able to visit yourname.github.io to see your website.

How to edit your index.html webpage file?

You can use my index.html as a template:

  1. Download the file from this link.
  2. Install Notepad++ and use it to edit the file.
  3. Replace all content (text, image names) with your own.
  4. Keep image files in the same folder as index.html.
  5. Upload index.html along with your image files to your GitHub repository.

Note: The RevolverMap code in index.html is person-specific. Generate your own code at RevolverMaps and replace the corresponding part in index.html (around line 305).
